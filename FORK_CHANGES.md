# Fork 改动说明

这份文件记录本 fork 相对上游版本的代码改动，供提交 Pull Request、代码审查和后续维护使用。新改动按日期倒序追加。

维护要求：

- 只记录已经实际完成、存在于仓库中的代码或文档改动。
- 不记录本机文件操作、模型目录调整、临时运行状态、个人数据状态或纯讨论内容。
- 用户可见的功能、界面、配置、错误处理、安装流程或兼容性发生变化时，同一轮任务必须更新本文件。
- 每条记录尽量说明“结果、涉及范围、验证方式、审查提示”，避免逐文件流水账。
- 正式合并到上游发布记录后，可以将对应条目移入 `CHANGELOG.md`，并在这里注明已归档。

## 2026-07-19

### 图片库完整中文化

- 图片库首页、筛选工作区、质量扫描、人物与风格分组、水印检测、重复图片处理、复制到数据集弹窗及共用文件夹选择器全部接入中英文语言包。
- 设置页新增的图片库质量筛选阈值同步接入语言包；新增契约测试，防止图片库主要入口重新出现硬编码英文。
- 验证：前端测试 332 项通过，生产构建成功；本地运行服务已加载新构建资源。

### 顶部导航自适应

- 桌面导航容器改为使用视口可用宽度并限制在 `2xl` 内容宽度内，英文等较长导航标签保持单行；移动折叠菜单维持原有 `md`（768px）切换点。
- Logo 与菜单保持相邻并作为一个整体水平居中，不使用弹性空白将两者推向容器两侧。
- 验证：前端测试 330 项通过，并完成生产构建。

### 语言下拉框配色

- 顶部语言选择器的原生下拉菜单改用不透明的深色浮层背景和深色系统控件主题，保留原有文字大小、宽度和高度。
- 验证：前端测试 328 项通过，并完成生产构建。

### 上游同步兼容

- 将本 fork 的中文化与可靠性改动重放到上游 `v2026.07.19.1` 最新主线，保留图片库、原生文件夹选择器、继续训练弹窗、完整备份和 ZIP 自动更新等上游新增功能。
- 新增图片库导航、图片库删除确认、双标注/继续训练帮助提示和 ZIP 更新进度的中英文文案，避免同步后相关入口回退为英文。
- 验证：前端测试 327 项通过；后端能力、数据集服务与设置接口测试 171 项通过。

### 简体中文界面与双语基础设施

- 新增英文和简体中文语言包、语言上下文与顶部语言切换按钮；浏览器语言为中文时默认使用简体中文，选择保存在 `lds.locale`。
- 语言包改为自动发现 `frontend/src/i18n/locales/*.json`；语言代码、原生名称、页面标题、文字方向和排序由文件内 `_meta` 描述，新增语言不再需要修改 React 注册代码。
- 顶部中英文二选一按钮改为自动读取所有语言包的多语言选择菜单，并支持按浏览器语言选择精确地区或默认地区语言包。
- 系统性汉化数据集工作区、生成镜头目录、采集、筛选与清理、标注、导入导出、训练、测试工作室、设置向导、设置各分区、帮助页、弹窗、按钮、提示和 Toast。
- 新增简体中文使用指南，覆盖入门、使用流程、设置参考、故障排查、获取帮助和数据集指南。
- 英文仍是回退语言；缺少中文键时不会显示空白文本。
- 主要入口：`frontend/src/i18n/`、`frontend/src/App.jsx`、`docs/guide/zh-CN/`。

### 前端错误恢复与诊断

- 完善全局错误页：显示实际错误消息、当前路由、堆栈和技术详情，并提供“重试”“返回数据集”“重新加载页面”和“复制诊断信息”。
- 调整局部变量命名，避免组件内部变量遮蔽翻译函数。
- 通用错误页、诊断报告、更新提示、帮助模式和通知组件接入双语文案。
- 图片拒绝/删除、设置向导返回、训练预设、续训、检查点回收、运行清理和强制上传等浏览器原生确认框全部接入语言包；新增契约测试，禁止原生确认框再次直接接收硬编码字符串。
- 主要入口：`frontend/src/components/common/ErrorBoundary.jsx`、`DiagnosticReport.jsx`、`Toast.jsx`。

### ComfyUI 生成失败立即停止整批

- 本地 Klein 批次出现首个真实 ComfyUI 错误时，当前图片会保留为失败并记录具体原因，剩余排队或运行中的同数据集任务立即取消，不再重复执行相同的错误工作流。
- 手动取消产生的回调不会再次触发整批取消，避免递归停止。
- 取消沿用精确任务/Prompt 中断，不使用无条件的全局 ComfyUI 中断。
- 数据集接口会自动校正失真的 Klein 活动计数，避免最后一个回调异常后生成状态永久不结束。
- 主要入口：`backend/app/job_queue.py`、`backend/app/services/face_dataset_service.py`。

### 生成停止入口

- 生成进行中时，镜头目录底部的主按钮会从紫色生成按钮切换为红色“停止生成”，并显示当前进度。
- 页面顶部生成提示条的停止按钮不再被全局 `busy` 状态错误禁用。
- 增加“正在停止”状态，防止重复点击，并为成功/失败结果提供双语通知。
- 主要入口：`frontend/src/components/dataset/VariationCatalog.jsx`、`DatasetWorkspace.jsx`、`frontend/src/hooks/useDataset.js`。

### 设置开关视觉修正

- 统一设置页药丸开关的定位方式：白色圆球使用固定的 `left/top` 基准，开启时右对齐，关闭时左对齐。
- 关闭状态改用内描边，避免边框参与尺寸计算造成圆球上下偏移一到两个像素。
- 已覆盖服务器局域网访问、访问令牌、训练相关开关等共享样式。
- 主要入口：`frontend/src/components/settings/ServerSection.jsx`、`TrainingSection.jsx`。

### Ollama 配置容错

- 保存和使用 Ollama URL 时自动清除首尾空格与末尾斜杠，避免复制地址时的隐形空格让正常服务被误判为不可访问。
- 能力检测、模型检测、启动控制、视觉请求、文本请求、卸载和模型下载统一使用规范化后的地址。
- 设置向导中的 Ollama/Qwen3-VL 状态和说明已汉化。
- 主要入口：`backend/app/capabilities.py`、`routes/settings.py`、`services/vision_ollama.py`、`services/ollama_control.py`。
