# 设置参考

设置页集中管理生成引擎、本地工具、训练、服务器和维护选项。

## 设置如何生效

普通设置在页面底部统一保存。密钥属于只写字段：保存后输入框仍为空，只通过“已配置”状态显示。

“测试”会先保存对应字段再检查连接。涉及服务器端口和监听地址的设置需要“保存并重启”才能生效。数据集专属设置位于数据集右上角的“更多 → 编辑设置”。

## 概览

概览页显示各项能力是否就绪，并提供对应配置入口。若首次使用，建议先运行设置向导。

## 图像引擎

- **Gemini API 密钥：** 用于 Nano Banana。
- **OpenAI API 密钥：** 用于 ChatGPT 图像 API。
- **ChatGPT 订阅：** 实验性连接，可使用 Plus/Pro 图像额度，受每日额度和 SFW 限制。
- **默认引擎：** 决定生成面板初始选择。
- **启用的引擎：** 控制生成面板显示哪些引擎。
- **Klein LoRA 预设：** 将自定义 LoRA 文件按顺序串联到本地 Klein 工作流。

密钥不会显示原值。移除密钥后，依赖该密钥的引擎会立即停止工作。

## 抓取与来源

这里保存 Reddit、Civitai、Pexels 等来源的可选凭据，并配置 Klein 小图修复指令。

Pexels API 密钥本身不等于数据集或机器学习用途授权；只有在 Pexels 明确批准用途后才应启用相应搜索。

## 本地工具

- **ComfyUI API 地址：** 通常是 `http://127.0.0.1:8188`。
- **ComfyUI 安装目录：** 用于寻找模型、LoRA、输入和输出目录。
- **Ollama：** 为自动标注、构图识别和头部裁剪提供本地视觉模型。
- **ai-toolkit：** 本地训练引擎；目录应包含 `run.py`，并具有可用虚拟环境。
- **Hugging Face 令牌：** 用于受限模型和私有仓库。

## 标注与质量

可选择自动标注后端，并配置：

- 水印修复使用自动、CUDA 或 CPU；
- 是否允许自动裁掉边缘水印；
- 人脸相似度绿色和橙色阈值；
- 人物蒙版和质量工具使用的 Python 环境。

阈值只影响提示和自动筛选，不会自动删除图片。

## 训练

设置新训练任务的默认模型系列，以及 vast.ai 云端训练限制：

- 最大并行任务数；
- 每小时最高价格；
- 月度预算；
- 无进度超时；
- 主机可靠性和安全云筛选。

预算为 `0` 表示不限制。降低可靠性阈值可能显示更便宜的主机，但启动失败风险更高。

## 服务器与访问

配置监听主机、端口、局域网访问和访问令牌。对局域网开放时务必设置令牌；修改监听设置后需要保存并重启。

## 维护

维护页提供：

- 当前版本与更新检查；
- 服务器日志查看和复制；
- 数据目录位置；
- 诊断报告；
- 清理缓存或检查运行状态。

诊断信息会隐藏密钥，但发送前仍应检查是否包含个人路径或文件名。

## 每个数据集的设置

在数据集“更多 → 编辑设置”中可以修改名称、类型、触发词、概念描述和生成提示词后缀。

更改数据集类型不会删除图片、标注或训练历史，但会改变标注策略和可见面板。已有标注不会自动重写；切换后应重新生成标注。

身体保真度仅适用于角色数据集。开启后，身体形状和永久标记也会绑定到触发词，并影响构图目标与导入裁剪策略。

## 仅配置文件可用的设置

少数部署级选项只在配置文件或环境变量中提供，例如数据根目录、日志级别和特定后端覆盖。修改前先停止服务并备份配置，不要把密钥提交到 Git。
