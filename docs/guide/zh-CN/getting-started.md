# 开始使用

LoRA 数据集工作室把参考图、图片筛选、自动标注、人脸评分和训练整合在同一个本地网页中。训练本身通常不是最难的部分；真正影响结果的是数据集是否干净、均衡且标注准确。

> **想快速开始？** 启动应用，让“设置向导”扫描本机，然后直接用自己的照片创建数据集。第一步不要求 API 密钥、GPU 或外部工具。

## 两种运行方式

| | 仅 API | 完整本地 |
|---|---|---|
| 可用功能 | 创建数据集、通过 Gemini/ChatGPT 生成、筛选、标注、导出 ZIP | 包含左侧全部功能，并增加 Klein 本地生成、人脸评分、蒙版、训练和测试工作室 |
| 需要 | Python 3.10–3.12、至少一个图像 API 或 ChatGPT 订阅 | ComfyUI 和/或 ai-toolkit，以及 NVIDIA GPU |
| 适合 | 笔记本、首次体验、云端训练 | 在训练工作站上完成全部流程 |

可以先使用 API 模式，以后再安装本地工具；应用检测到工具后会自动开放相应功能。

## 首次启动

**Windows：** 下载最新发布包，解压后双击 `start.bat`。脚本会寻找或下载兼容的 Python，创建 `.venv`，安装依赖并打开 `http://127.0.0.1:5050/`。

**手动虚拟环境：**

```text
python -m venv .venv
.venv\Scripts\activate
pip install -r backend/requirements.txt
python backend/run.py
```

**Docker（仅 API 模式）：** 复制 `.env.example` 为 `.env`，然后运行 `docker compose up --build`。

## 设置向导

首次启动时，设置向导会扫描本机并依次检查：

1. **图像生成**——配置 Gemini、OpenAI/ChatGPT 或本地 Klein。
2. **ComfyUI**——启用 Klein 本地生成和测试工作室。
3. **Ollama**——提供自动标注、构图识别和头部裁剪。
4. **质量工具**——安装人脸相似度、人物蒙版和水印修复依赖。
5. **ai-toolkit**——启用本地 LoRA 训练。

所有步骤都可以跳过，并可随时在“设置”中重新配置和测试。

## 应用导航

- **数据集**——创建并管理数据集，依次完成图片来源、筛选、标注、训练。
- **训练任务**——查看本地和云端训练进度、日志、重试与下载结果。
- **测试工作室**——比较不同检查点、强度和生成设置。
- **指南**——当前使用手册。
- **设置向导**——重新扫描并配置本机工具。
- **设置**——管理引擎、密钥、本地工具、训练和服务器选项。
- **帮助**——生成诊断信息并查找反馈渠道。
